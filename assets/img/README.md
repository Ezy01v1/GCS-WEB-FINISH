## Imágenes

Coloca aquí tus imágenes, por ejemplo:

- `assets/img/hero.jpg`
- `assets/img/logo.png`
- `assets/img/campus-1.jpg`

### Cómo usarlas en el HTML

Ejemplo:

```html
<img src="assets/img/hero.jpg" alt="Galilee Christian School">
```

### Cómo usarlas desde el CSS

Ejemplo (en `assets/css/styles.css`):

```css
#hero {
  background-image: url("../img/hero.jpg");
  background-size: cover;
  background-position: center;
}
```
